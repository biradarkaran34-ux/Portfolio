
# Registration Project
![HTML](https://img.shields.io/badge/HTML-5-orange)
![CSS](https://img.shields.io/badge/CSS-3-blue)


## Description
This project is a simple **registration form** built using HTML and CSS.  
It allows users to enter their details such as name, email, phone number, date of birth, gender, password, and skills.  
The form is styled with a modern dark theme and teal accents for a clean look.


---

## 🚀 How to Run
1. Open `project.html` in any web browser (Chrome, Edge, Firefox, etc.).
2. The form will automatically load styles from `project.css`.

---

## 🎨 Features
- Dark navy background with teal highlights.
- Input fields for:
  - Full Name  
  - Email  
  - Phone Number  
  - Date of Birth  
  - Gender (radio buttons)  
  - Password & Confirm Password  
  - Skills  
- Styled button with hover effect.
- Responsive and clean design.

---

## 👥 Target Audience
This project is for:
- Beginners learning **HTML & CSS**.
- Students who want to practice building forms.
- Developers who want a base template for registration/admission forms.
